# Variables from the Decennial censuses

Variables from the Decennial censuses

## Usage

``` r
DECENNIAL_POPULATION_FIELDS
```

## Format

A tibble with five columns

- Year:

  The year of the decennial census.

- Group:

  The table that the variables come from.

- Race/Ethnicity:

  The OMB label for a race/ethnic group.

- Index:

  The row in the table that corresponds to the race/ethnicity.

- Variable:

  The name of the variable for an API call
