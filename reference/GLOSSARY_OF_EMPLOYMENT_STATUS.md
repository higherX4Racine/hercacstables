# Factor values associated with specific rows within employment-related ACS tables

Factor values associated with specific rows within employment-related
ACS tables

## Usage

``` r
GLOSSARY_OF_EMPLOYMENT_STATUS
```

## Format

### GLOSSARY_OF_EMPLOYMENT_STATUS

A data frame with 439 rows and 12 columns:

- group:

  `<chr>`

- index:

  `<int>`

- variable:

  `<chr>`

- Sex:

  `<chr>`

- Lower Age:

  `<int>`

- Upper Age:

  `<int>`

- Labor Force:

  `<chr>`

- Enlistment:

  `<chr>`

- Employment:

  `<chr>`

- Suffix:

  `<chr>`

- Census Race/Ethnicity:

  `<chr>`

- Poverty:

  `<chr>`

## Source

api.census.gov/data/acs/acs5/groups.html
