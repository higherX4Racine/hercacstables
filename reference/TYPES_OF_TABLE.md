# A Census table's ID always starts with an alphanumeric code for its type.

A Census table's ID always starts with an alphanumeric code for its
type.

## Usage

``` r
TYPES_OF_TABLE
```

## Format

### TYPES_OF_TABLE

A data frame with 11 rows and 3 columns.

- Table ID Characters:

  `<chr>` The alphanumeric code for this type of table

- Type of Table:

  `<chr>` A human-readable name for the type of table.

- Contains:

  `<chr>` A description of what information this type of table will
  contain.
