# ACS data contain variables ending in 1- or 2-letter codes.

This table defines each variable type and its meaning.

## Usage

``` r
TYPES_OF_VARIABLE
```

## Format

### TYPES_OF_VARIABLE

A data frame with 7 rows and 4 columns.

- Variable Type:

  `<chr>` A 1- or 2-letter code

- Title:

  `<chr>` A human-readable name for the type of variable.

- Related Products:

  `<chr>` The types of Census data that contain this kind of variable.

- Meaning:

  `<chr>` A detailed explanation of what this kind of variable
  represents.
