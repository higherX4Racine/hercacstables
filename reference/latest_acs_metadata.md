# Download glossary about the most recent versions of the 1- and 5-year ACS

Download glossary about the most recent versions of the 1- and 5-year
ACS

## Usage

``` r
latest_acs_metadata(information_type)
```

## Arguments

- information_type:

  \<chr\> "geographies", "groups", or "variables"

## Value

a tibble with dimensions that depend upon `information_type`
