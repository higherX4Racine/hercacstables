# Today's year in the Gregorian calendar.

Today's year in the Gregorian calendar.

## Usage

``` r
the_year_right_now()
```

## Value

\<int\> probably 2024 or later

## See also

[`as.POSIXlt()`](https://rdrr.io/r/base/as.POSIXlt.html),
[`Sys.Date()`](https://rdrr.io/r/base/Sys.time.html)

## Examples

``` r
the_year_right_now()
#> [1] 2026
```
